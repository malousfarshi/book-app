# Book App

Android application built with Kotlin and Jetpack Compose to display a list of books.  
Data is fetched from a mock API.

## Features
- Fetch book data from a remote API
- Display list of books with Compose
- Basic UI with customization options
- Offline usage (local cache planned)
- Move history shows user interactions (optional, if implemented later)

## Technical Details
- Built with Kotlin and Jetpack Compose
- Networking with Ktor HttpClient
- JSON parsing with kotlinx.serialization
- BASE_URL: Mock API endpoint
- Uses @OptIn for experimental API features


## Status
- Project is under development
- Improvements and additional features ongoing
